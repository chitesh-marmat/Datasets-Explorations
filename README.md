# Dataset Explorations

Welcome to the **Dataset Explorations** repository! In this project, I explore a variety of datasets, clean them, and perform **Exploratory Data Analysis (EDA)**. The goal of this repository is to leave the dataset ready for further model building and explorations, allowing anyone to pick it up and start building different types of machine learning models.

## 📂 Project Structure

The repository contains different Jupyter notebooks, each dedicated to a different dataset.
